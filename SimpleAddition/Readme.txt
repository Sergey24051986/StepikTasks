1. Программа устанавливает по щелчку левой кнопки мыши компонент
   (увеличивающиеся, а потом уменьшающиеся круги)
2. Есть возможность при нажатии правой кнопки мыши изменить режим
    (поменять полые круги на заполненные)
3. Все компоненты помещаются поверх следующего!
4. В коде оставил закомментированный режим
   (шар лопается, а не уменьшается)
