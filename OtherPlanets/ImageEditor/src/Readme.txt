Программа ImageEditor создана дя обработки изображения, точнее настойки области изъятия изображения в виде круга, овала, квадрата и прямоугольника.
Присутствует файл Start.bat для автономного запуска без компилятора (в ней нужно настроить путь к файлу ImageEditor.java)
Для загрузки изображения нужно ввести в поле "Загрузить" только название файла из папки Original - например ввести "EarthOrig".
Для сохранения в поле "Сохранить" тоже только название файла, например ввести "Earth".
