Калькулятор. Финал.

Продолжите работу над проектом. Теперь в файле input.txt лежит не одна, а неизвестное количество строк (пример строки
описан в задаче Простой калькулятор). Ваша задача - сформировать файл output.txt, в котором для каждой строчки будет
лежать ответ вида: "<исходная строка> = <ответ к этой строке>". В качестве ответа пришлите ссылку на git - репозиторий
вашего проекта.

Пример:

input.txt	output.txt

23 + 42     23 + 42 = 65.0

14 - f      14 - f = Error! Not number

15 / 0      15 / 0 = Error! Division by zero

45 * 2      45 * 2 = 90.0
