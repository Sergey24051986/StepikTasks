Программа ImageEditor создана дя обработки изображения, точнее настойки области изъятия изображения в виде круга, овала, квадрата и прямоугольника.
Присутствует файл Start.bat для автономного запуска без компилятора (в нем нужно настроить путь к файлу ImageEditor.java)

Функционал:
- Можно настраивать область мышкой и перемещать ее
- Колесиком мышки можно расширять и уменьшать область
- Для более тонкой настройки есть кнопки
